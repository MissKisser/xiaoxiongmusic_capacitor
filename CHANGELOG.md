# 更新日志

## 4.6.6 - 2026-06-08

### 新增

- 新增 Android 桌面歌词原生悬浮窗，支持系统悬浮窗权限、歌词同步、拖动定位、锁定穿透和关闭。
- 全屏播放器播放设置中新增桌面歌词设置入口，通知栏播放器和播放页快捷入口支持一键开关桌面歌词。
- 桌面歌词工具栏新增字号调整、颜色预设、上一曲、播放/暂停、下一曲、锁定和关闭操作。

### 优化

- 移除桌面歌词逐字高亮，统一为主流桌面歌词的整句显示体验。
- 点击未锁定桌面歌词时显示整块半透明交互遮罩，设置中不再暴露遮罩开关。
- 展开工具栏、歌曲名和颜色面板时保持主歌词锚点不动，避免歌词上下闪动。
- 颜色预设改为折叠面板，减少工具栏常驻按钮数量。

### 修复

- 修复桌面歌词点击后遮罩不显示的问题。
- 修复通知栏播放器缺少桌面歌词快捷按钮的问题。
- 修复桌面歌词工具栏显示时歌曲名挤压歌词位置的问题。

### 验证

- `pnpm exec tsx scripts/test-android-desktop-lyric.ts`
- `pnpm build`
- `pnpm cap:sync:android`
- `./gradlew.bat :app:assembleRelease`
